# 💬 WebSocket Chat App

A simple real-time chat application built with **Node.js**, **Express**, and **WebSocket** (ws). Messages are broadcast to all connected users and stored in memory as chat history, shown to users when they join.

---

## ⚙️ Tech Stack

- Node.js
- Express
- ws (WebSocket)
- HTML, CSS, JavaScript
